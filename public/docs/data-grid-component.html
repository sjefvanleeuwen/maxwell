Data Grid Web Component

Introduction

The Data Grid component is a lightweight, customizable HTML table component that can display structured data in a tabular format. It's built using Web Components, making it framework-agnostic and easy to integrate into any web application.

Features

  • Framework-independent
  • Simple data structure
  • Customizable styling
  • Supports dynamic data updates
  • Separated template structure

Installation

Import the component in your HTML file:

<script type="module" src="/components/data-grid/data-grid.js"></script>

Basic Usage

Add the component to your HTML:

<data-grid id="myGrid"></data-grid>

Initialize with data:

const grid = document.getElementById('myGrid');
grid.setData({
    headers: ['Name', 'Age', 'City'],
    rows: [
        ['John Doe', 30, 'New York'],
        ['Jane Smith', 25, 'London']
    ]
});

API Reference

Methods

  • setData(data): Updates the grid with new data

Data Structure

{
    headers: string[],  // Array of column headers
    rows: any[][]      // Array of row data
}

Customization

You can customize the appearance of the grid using CSS. The component uses standard table elements that can be styled using CSS selectors:

data-grid table { /* styles */ }
data-grid th { /* styles */ }
data-grid td { /* styles */ }
